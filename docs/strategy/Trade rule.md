# Trade rule

## 対象スタイル

AI・半導体関連の大型株・テーマ内リーダー株を中心に、信用取引で2〜5銘柄へ絞ってスイングする運用を対象にしたルール。

## 1. 資金管理

- 運用資金1,000万円を前提に、平時の最大建玉総額は1.25倍〜1.5倍に抑える
- トレードが連勝しており、相場全体も強い上昇トレンドにある場合に限り、最大建玉総額を2倍まで引き上げる
- 損失が出始めた時、ブレイクアウトのダマシが増えた時、地合いが崩れた時は、レバレッジを即座に1倍以下へ落とす
- レバレッジは勝っている時だけ使い、守りに入る局面では現金比率を優先する

## 2. 1銘柄あたりの上限

- 1銘柄あたりの最大ポジションは、自己資金の25%〜50%を上限とする
- 1,000万円運用なら、1銘柄あたり250万〜500万円が建玉の目安
- 2〜5銘柄に絞る場合でも、同じ値動きしかしない銘柄を重複保有しない

## 3. 信用と現物の使い分け

- 本命銘柄でも、モメンタムが継続している限りは信用のまま保有してよい
- 1か月半程度の保有なら、信用金利は強いリーダー株の上昇余地に対して必要経費として扱う
- 株価が高値を切り上げている間は、現引を前提にせず、信用枠の柔軟性を優先する
- ただし、短期の入れ替え余地を失うほど信用枠を固定しない

## 4. 銘柄選定

- AI・半導体テーマ全体に広く分散するのではなく、今いちばん資金が入っているサブグループの絶対的リーダーへ寄せる
- 「強い領域の出遅れ銘柄を拾う」発想ではなく、「先導している上位1〜2銘柄へ寄せる」発想を優先する
- 分散のためだけの分散はしない
- ただし、同じサブグループの銘柄を複数持つ重複は避ける
- 持つなら、AIテーマ内でも異なるサブグループのリーダーを2〜3銘柄へ絞る

## 5. 損切りルール

- 買値から最大7%〜8%下落したら、理由を付けず機械的に損切りする
- レバレッジをかけている局面では、このストップロスを例外扱いしない
- 急落リスクは銘柄数を増やして薄めるのではなく、損切りの速さと現金確保でコントロールする

## 6. 利確ルール

- 買値から20%〜30%上昇したら、そのトレードを損で終わらせない前提へ切り替える
- その時点でストップ位置を買値まで引き上げる
- さらに上昇したら、10日線や25日線、直近サポートの下へトレイリングストップを切り上げて利益を守る
- 利益はできるだけ伸ばしつつ、含み益が大きく乗った玉から守りを固める

## 7. 決算跨ぎ

- 含み益が十分にない状態で、信用フルポジションのまま決算を跨がない
- 決算前に最低でも10%〜15%以上の含み益がないなら、半分〜全部を利確または縮小する
- クッションが不足している時は、決算後の値動きを見てから入り直す
- 決算は例外なくイベントリスクとして扱い、強いリーダー株でもギャンブルにしない

## 8. 地合い悪化時の対応

- 悪材料による窓開け急落は、逆指値を入れていても想定以上の損失になる前提で構える
- 機関投資家の売り抜けサインが増えた時、リーダー株が異常な急騰を見せた後、相場全体の上昇トレンドが弱った時は、ポジションサイズを半分以下へ落とす
- 危険を感じる局面ではレバレッジを解除し、身軽さを優先する

## 9. 運用の優先順位

1. 最優先は退場しないこと
2. そのために、最初に徹底すべきなのは資金管理
3. その上で、強いテーマのリーダー株へ資金を寄せる
4. 利益は大きく伸ばし、損失は初動で切る

## 要点

- 集中投資のリスクは、分散の量ではなく、建玉上限・ストップロス・現金比率で管理する
- 信用取引は資金効率を高める手段だが、勝っている局面でだけ使う
- ルールを破って大きく負けないことが、次のチャンスを取り続ける前提になる
