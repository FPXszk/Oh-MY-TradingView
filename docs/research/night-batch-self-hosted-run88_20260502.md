   vol20x13（PF=3.806, DD=1,427）と vol20x12（PF=3.662, DD=1,542）は vol 閾値のみ異なる。閾値 vol20x10〜vol20x20 を固定ステップで変えた系列を作成し、PF・DD 両立の変曲点を特定する。目安: vol20x13〜vol20x15 付近に最適帯が存在するか確認。

2. **MSTR 依存リスクの定量評価**
   vol20x13（MSTR 52.8%）と combo-ema50200（MSTR 56.4%）は MSTR に過半依存。MSTR を除外したサブセット（39銘柄）で avg_net_profit と avg_profit_factor を再集計し、依存排除後も PF ≥ 2.0 を維持できるか確認する。

3. **PF 最優秀の vol20x20 が低 net な原因特定**
   emr-next-vol20x20 は PF=4.559 と全戦略最高だが avg_net_profit=804.46（全戦略最低水準）。trade 数が少ないことが原因と推定されるため、平均 trades 数を vol20x13 と比較し、エントリー機会の損失量を定量化する（vol20x13 は 40 銘柄で大量トレード、vol20x20 は高閾値で機会激減）。

4. **ATR 系ストップの除外確定検証**
   atr15x（PF=1.383）・atr20x（PF=1.444）は即除外だが、atr25x（PF=1.514）は要注意圏。ATR 倍率を 30x・35x まで拡張した場合に PF が 1.7 を超えるか 2 run で検証し、超えなければ ATR 系ストップ全廃を確定する。

5. **次回比較の固定指標**
   次回 run との比較は avg_net_profit / avg_profit_factor / avg_max_drawdown / MSTR 集中度(%) の 4 指標のみで行い、composite_score は参考値とする。avg_win_rate は本戦略群では評価対象外（全 50 戦略が 35% 未満のため）。
