## 調査内容

### TradingView Scanner API の確認

- **エンドポイント**: `POST https://scanner.tradingview.com/america/scan`
- **認証**: 遅延データは不要（公開 API として機能）
- **フィールド数**: 3,000+ のフィールドが利用可能
- **実装参考**: `shner-elmo/TradingView-Screener` の Python パッケージが同一 API を使用

### 正しいフィールド名の確認（`curl` で直接検証）

| 意図 | 誤ったフィールド名 | 正しいフィールド名 |
|---|---|---|
| 相対出来高 | `Relative.Volume` | `relative_volume_10d_calc` |

`Relative.Volume` を使うと `Unknown field` エラーが発生した。`curl` で実際に叩いて修正済み。

### 動作確認済みフィールド一覧

`RSI`, `SMA200`, `SMA50`, `EMA20`, `MACD.macd`, `MACD.signal`, `market_cap_basic`, `Perf.W`, `Perf.1M`, `Perf.3M`, `Perf.6M`, `Perf.Y`, `close`, `price_52_week_high`, `price_52_week_low`, `average_volume_90d_calc`, `relative_volume_10d_calc`, `earnings_per_share_diluted_ttm`, `dividends_yield_current`

### 使用できないエンドポイント（403）

- `symbol-search.tradingview.com`
- `economic-calendar.tradingview.com`

### バックテストについて

外部から取得可能な API なし。CDP + TradingView Desktop の Strategy Tester が必須。

## ミネルビニ7条件と実装方式

| # | 条件 | 実装方式 |
|---|---|---|
| 1 | 対象市場：米国株のみ | API `markets: ["america"]` |
| 2 | RSI(14) ≥ 60 | サーバー側フィルタ |
| 3 | 時価総額 ≥ 10億ドル | サーバー側フィルタ |
| 4 | 直近出来高 ≥ 平均の1.2倍 | サーバー側フィルタ |
| 5 | 現在値 > 200日移動平均線 | クライアント側後処理 |
| 6 | 現在値 > 50日移動平均線 | クライアント側後処理 |
| 7 | 現在値 ≥ 52週高値の75% | クライアント側後処理 |

SMA が null の銘柄（OTC 等）はそのフィルタをスキップして通過する（意図的設計）。
