# Pine references

このディレクトリは **Pine source snapshot の保管庫** です。  
上位戦略を chart review するときの durable copy をここに置きます。

## 目的

- ranking artifact と対応づく Pine source を固定する
- 人手レビュー時に「どの source を見たか」を追えるようにする
- chart 適用前の確認用 snapshot を残す

## 補足

- strategy lifecycle の正本ではありません
- lifecycle / 説明文の正本は `config/backtest/strategy-catalog.json`
- 人間向けの入口は `docs/research/strategy/README.md`
